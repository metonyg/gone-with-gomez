# Product Context

The site exists to tell a daily illustrated family adventure in a format that feels like an unfolding serial.

The reader experience should be warm, visual, and easy to navigate. Each page should feel self-contained while also advancing the 365-day arc.

Images should match the story's animated family-adventure tone and keep the Gomez family visually recognizable over time.

Current image consistency is prompt-based, not true reference-image conditioning.
